# DevOps on AWS Workshop

** Date:** Monday, November 17, 2025, 8:30 AM  5:00 PM  
** Location:** AWS Vietnam Office  
** Role:** Attendee

## Event Overview
Comprehensive full-day workshop covering AWS DevOps services including CI/CD pipelines, Infrastructure as Code, container services, and monitoring & observability.

## Key Highlights
- DevOps culture, principles, and DORA metrics (MTTR, deployment frequency)
- CI/CD pipeline: CodeCommit, CodeBuild, CodeDeploy with Blue/Green, Canary, Rolling updates
- Infrastructure as Code: CloudFormation templates vs AWS CDK
- Container services: Docker, ECR, ECS, EKS, App Runner
- Monitoring: CloudWatch metrics/logs/alarms, AWS X-Ray distributed tracing
- DevOps best practices: feature flags, automated testing, incident management
- Case studies: Startup and enterprise DevOps transformations
- Career pathways and AWS certification roadmap

## Key Takeaways
 CI/CD automation reduces deployment time  
 IaC enables reproducible infrastructure  
 Containers provide consistency across environments  
 Observability crucial for production systems  
 DevOps culture drives business agility
