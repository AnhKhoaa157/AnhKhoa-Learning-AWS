# AI/ML/GenAI on AWS Workshop

** Date:** Saturday, November 15, 2025, 8:30 AM  12:00 PM  
** Location:** AWS Vietnam Office  
** Role:** Attendee

## Event Overview
Hands-on workshop covering Amazon SageMaker for traditional ML workflows and Amazon Bedrock for generative AI applications, including RAG architecture, prompt engineering, and Bedrock Agents.

## Key Highlights
- Amazon SageMaker: data preparation, model training, deployment, MLOps
- Generative AI with Amazon Bedrock: foundation models comparison
- Prompt engineering: Chain-of-Thought reasoning, Few-shot learning
- RAG architecture and knowledge base integration
- Bedrock Agents for multi-step workflows
- Guardrails for safety and content filtering
- Live demo: Building generative AI chatbot

## Key Takeaways
 End-to-end ML platform with SageMaker  
 Multiple foundation models via Bedrock  
 RAG improves response accuracy  
 Prompt engineering critical for LLM performance  
 Guardrails ensure safe AI applications
